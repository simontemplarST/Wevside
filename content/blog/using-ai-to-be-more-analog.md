# 

I've gone down this road many times before, trying to become a luddite outside of work. 
I've come to the realization that tech is going to continue to integrate into every day life.
It already has for all intents and purposes. 

I've made a couple things this weekend using AI and it has allowed me to do a full delete of streaming
services and even my own local media server stack. All I have is a small NAS, an Apple TV, and a few 
scripts that interact with the NAS' API.
(I have a thing for TUI's so I can SSH into home and get things done)
First up is Curate. It's essentially a simple file manager with a LOT of Metadata editing features.
It has the ability to download metadata from TVDB and TMDB, and inject it directly into the file.
